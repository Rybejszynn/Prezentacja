Darmowy Hosting Stron na GitHub

Przewodnik krok po kroku: Jak opublikować swoją witrynę w kilka minut bez żadnych kosztów.

Co to jest GitHub Pages?

Szybkie i niezawodne rozwiązanie do hostowania statycznych witryn bezpośrednio z Twojego repozytorium kodu.

Dlaczego warto wybrać GitHub Pages?

100% Za Darmo

Brak opłat abonamentowych, kosztów serwera czy limitów transferu dla małych projektów.

Bezpieczeństwo

Automatyczny certyfikat SSL (HTTPS) zapewnia bezpieczne połączenie dla Twoich użytkowników.

Własna Domena

Możliwość podpięcia profesjonalnego adresu (np. moja-strona.pl) w kilka minut.

Zaufanie Milionów

100M+
Aktywnych Projektów

Największa Społeczność

GitHub to standard branżowy. Korzystając z niego, dołączasz do największej na świecie platformy dla programistów i twórców.

Proces Publikacji

Trzy proste kroki, które dzielą Cię od własnego miejsca w sieci.

Czego potrzebujesz na start?

Konto GitHub

Darmowe konto na portalu github.com to Twój bilet do hostingu.

Pliki Strony

Twoje pliki HTML, CSS i JS. Pamiętaj o pliku index.html!

5 Minut

Tyle czasu zajmuje konfiguracja od zera do działającej strony.

Krok 1: Nowe Repozytorium

  • Kliknij ikonę plusa i wybierz "New repository".
  • Nadaj nazwę: nazwa-użytkownika.github.io.
  • Ustaw widoczność na Public.
  • Kliknij zielony przycisk "Create repository".

Krok 2: Dodaj swoje pliki

Przygotuj pliki

Upewnij się, że główny plik nazywa się index.html.

Wgraj je

Użyj opcji "uploading an existing file" i przeciągnij pliki do przeglądarki.

Zatwierdź

Kliknij "Commit changes", aby zapisać pliki w chmurze.

Krok 3: Aktywacja

  • Wejdź w zakładkę Settings repozytorium.
  • W menu bocznym wybierz Pages.
  • W sekcji "Branch" wybierz main i kliknij Save.
  • Twoja strona jest gotowa pod adresem twoja-nazwa.github.io!
